Oversampling¶

First we create another X and y with the oversampled data. To balance the dataset, we use the SMOTE algorithm. It will generate fake data with the label which is less in count - in this case the malicius data.

Undersampling¶

We create another X and y, in this case with undersampled data. We use the resample function from sklearn. It will balance the data by deleting the rows with higher count until the label count is even.
